✅ 1. Single Inheritance

A single class inherits from another class.

Example:

class Parent {
    void show() {
        System.out.println("Parent class");
    }
}

class Child extends Parent {
    void display() {
        System.out.println("Child class");
    }
}


Explanation: Child inherits all properties and methods of Parent.

✅ 2. Multilevel Inheritance

A class inherits from a child class, forming a chain.

Example:

class GrandParent {
    void method1() { System.out.println("GrandParent"); }
}

class Parent extends GrandParent {
    void method2() { System.out.println("Parent"); }
}

class Child extends Parent {
    void method3() { System.out.println("Child"); }
}


Explanation: Child indirectly inherits from GrandParent through Parent.

⚠️ 3. Multiple Inheritance (via Interfaces only)

Not allowed with classes (to avoid ambiguity),
but allowed with interfaces because interfaces only contain method declarations (no conflict in implementation).

Example:

interface A {
    void show();
}

interface B {
    void show();
}

class C implements A, B {
    public void show() {
        System.out.println("Multiple inheritance via interfaces");
    }
}

✅ 4. Hierarchical Inheritance

Multiple classes inherit from the same parent.

Example:

class Parent {
    void show() { System.out.println("Parent"); }
}

class Child1 extends Parent {}
class Child2 extends Parent {}


Explanation: Both Child1 and Child2 share common properties from Parent.

⚙️ 5. Hybrid Inheritance

A combination of two or more types (like hierarchical + multiple).

Not directly supported in Java using classes, but possible using interfaces.

Example (with interfaces):

interface A { void a(); }
interface B extends A { void b(); }
interface C { void c(); }

class D implements B, C {
    public void a() {}
    public void b() {}
    public void c() {}
}
